## Course

### Install

You need npm, docker and make in order to build it.

```bash
make
```

### Run

```bash
docker run -d --name cours vibioh/md-genie-logiciel
```

Browse [LocalHost](http://localhost:1080)

## Java sample

```bash
mvn clean install
```
